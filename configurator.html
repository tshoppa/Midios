BASIC OPERATIONS

The interface shows the Midimix

  • knobs,
  • buttons and
  • faders
For each controller click on
  • CC to edit the continuous controller message (0 - 127)
  • CH to edit the midi channel (1-16)
  • VL to edit the velocity value to be send (buttons only) (0 - 127)
  • buttons CC label to switch between CC and Note (NT) message
Values outside the possible range will be reset to the former value.

Use LEFT and RIGHT buttons to switch between the two configuration sets.

BUTTON CONFIGURATION

Buttons fire ON and OFF events:

  • ON sends the configured CC or Note with value VL on midi channel CH
  • OFF sends the configured CC with value 0 on midi channel CH or a note off message for Notes
In general, the button LED is lit from sending ON to sending OFF. Different button types define when and how events are fired. To edit a button type, click on the button icon. Available types are
PUSH
The default button type. Sends ON when press and OFF on release. LED is lit while pressed. Use Push buttons for "fire and forget" actions like Stop or Rewind.
TOGGLE
switches between the states ON and OFF when pressed repeatedly. Sends ON when pressed once and OFF when pressed again. The LED is lit while the state is ON. Toggle buttons are useful for actions like Rec on/off or Play/Pause
RADIO
combines multiple buttons in a group. In a radio group, only one button can be active. Pressing a group's button will activate that button and deactivate the formerly active. Sends button's ON when activated and OFF when deactivated. The active button's LED is lit. Radio groups are used for mutual exclusive state like selected track
TOGGLE GROUP
behave like normal toggle buttons, but can be operated all at once using a corresponding TOGGLE ALL button.
TOGGLE ALL
toggle all buttons of the associated toggle group. If some buttons are ON and some are OFF, all buttons will be switched to ON first. A second push will set all buttons to OFF. Use it to create Mute all or Solo all functions.
You can define up to 8 radio and toggle groups, each containing up to 8 buttons. A group can be used on one bank only. Groups that reached the maximum or are used on the other bank are disabled (grayed out) in the menu. A tooltip will give an indication of the reason.

UPLOAD and DOWNLOAD

Hit CREATE to save your configuration. The download will start automatically or you will be prompted a "Save As" dialog (depending on your browser configuration).

To upload a configuration, click on the config file link or simply drag and drop the file on the configurator. StreamByter files usually have a .sbr file ending.

Tipp: You can also drag links to .sbr configurations onto the configurator to load them! Try it with this example file!

TRACK DOUBLES

With over 100 controlls on two banks you might loose track on what is set where and unintentionally assign the same values to two or more different controllers. The track double function will mark controlls that use the same CC and CH combination (For buttons it will also consider the type CC or NOTE).

  • Click on TRACK DOUBLE to activate tracking mode.
  • Click on any knob, button or fader icon to select the controller. All controllers with the same CC and CH value will be highlighted. Buttons must also match the type (CC or Note).
  • Switch the bank to track doubles on the other bank
  • You can still change the controlls CC and CH values. Highlighting will be adopted accordingly.
  • Click on TRACK DOUBLE again to return to configuration mode.
